How it works

A domain-agnostic ingestion + retrieval pipeline designed for production.

1) Ingest & normalize

Connect your mailbox (IMAP), sync manual or automatic, extract body + attachments, and normalize into a canonical model.

  • Canonical entities: Document, Artifact, Chunk, Relations
  • Universal metadata: source, time, author, type, language
  • Deduplication: Message-ID + normalized fingerprint fallback

2) Structure progressively

Build lightweight hierarchy and summaries over time — without domain-specific assumptions.

  • Section/page hierarchy when available
  • Email ↔ attachment relationships
  • Summaries per section and per document

3) Index multi-representation

Multiple search views for higher recall: chunk, document, title/subject, and sparse tokens for rare identifiers.

  • Chunk embeddings + doc-level embeddings
  • Title/subject embeddings
  • Sparse representation for exact matches

4) Retrieve, rerank, cite

Candidate generation → reranking → consolidation → grounded response with citations (or explicit “not found”).

  • Hybrid candidate generation
  • Lightweight reranking for precision
  • Citations and confidence signals

# Illustrative API shape (example)
POST /v1/ingest
{
  "tenant_id": "t_123",
  "source": { "type": "imap", "mailbox": "INBOX" },
  "mode": "incremental"
}

POST /v1/search
{
  "tenant_id": "t_123",
  "query": "termination notice clause",
  "filters": { "source_type": ["email","pdf"], "date_from": "2024-01-01" },
  "k": 25
}

POST /v1/answer
{
  "tenant_id": "t_123",
  "query": "What is the termination notice period?",
  "policy": { "require_citations": true, "allow_not_found": true }
}

Trust and control

Built for production realities: isolation, traceability, and conservative behavior.

Isolation by tenant

Collections and storage are segmented per tenant, preventing cross-tenant leakage and enabling clean compliance boundaries.

Auditability

Every sync run is a job with status, progress, errors, and cost signals — designed to debug production issues fast.

Conservative answers

Grounding is mandatory. If evidence is missing, Vixai returns “not found” instead of guessing.
